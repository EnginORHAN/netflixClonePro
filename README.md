
#  Netflix Clone Project

## Özet
Bu proje, Netflix'in web sitesinin bir klonunu HTML, CSS, Bootstrap, Python ve Django kullanarak oluşturmayı amaçlamaktadır. Kullanıcılara Netflix'e benzer bir kullanıcı arayüzü ve işlevselliği sunmayı amaçlamaktadır.

## Özellikler
-  **Giriş, Kayıt ve Şifremi unuttum** Kullanıcıların Netflix'e giriş yapmalarına veya yeni bir hesap oluşturmalarına izin verir.
-   **Film ve Dizi Detay Sayfaları:** Filmler ve diziler hakkında ayrıntılı bilgi içerir, fragmanları oynatır ve benzer içerikler önerir.
- **Kullanıcı Profili:** Kullanıcıların hesap bilgilerini ve ayarlarını yönetmelerine olanak tanır.
- **Profil Düzenlemeleri:** Kullanıcı hesap biligilerini sonradan düzenleyebilir.
